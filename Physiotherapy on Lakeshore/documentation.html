Documentation

Introduction

EvoTemplate is a simple, clean, beautiful and fully responsive Bootstrap 3 Template for any business or personal use. Our code is very organized and easy to customize. You can use it on all main browsers, and also on your phone or tablet.

Evotemplate is fully animated and very fluid. You just have to scroll down to view the page. It also comes with some separate pages like, 404 error, portfolio item, FAQ and more.

We assume that you are already familiarized with bootstrap documentation, so this guide is not very extensive. If you want to know more about Bootstrap go here.

We will update our template in the future and we will listen to our customers for opinions and suggestions of new features.

Main Structures

Home Page Main Structures are: nav, header, wrapper and footer.

Page Beginning

Add these tags at the beginning of the file.

						<!DOCTYPE html>
						<html lang="en">

Page Head

Page head contains title, metadata and css includes.

						<meta charset="utf-8">
							<meta http-equiv="X-UA-Compatible" content="IE=edge">
							<meta name="viewport" content="width=device-width, initial-scale=1">
							<meta name="description" content="Very Clean and Modern Template for your business or personal website">
							<meta name="author" content="evoeffect">

							<title>EvoTemplate - Clean Teamplate</title>

							<!-- Bootstrap Core CSS -->
							<link href="css/bootstrap.min.css" rel="stylesheet">

							<!-- Custom CSS -->
							<link href="css/style.css" rel="stylesheet">

							<!-- Animation CSS -->
							<link href="css/animate.min.css" rel="stylesheet">

							<!-- Hover Effects CSS -->
							<link href="css/hover-min.css" rel="stylesheet" media="all">

							<!-- Custom Fonts -->
							<link href="fonts/font-awesome-4.2.0/css/font-awesome.min.css" rel="stylesheet" type="text/css">

							<!-- PrettyPhoto -->
							<link rel="stylesheet" type="text/css" href="css/prettyPhoto.css" />

Navigation

Navigation contains menu, logo and sub menus.

						<!-- Navigation -->
						<nav class="navbar navbar-inverse navbar-fixed-top" role="navigation">
							...
							...
						</nav>
						<!-- End of Navigation -->

To add a menu or sub menu item you have to add the following line: (you have to change the id and the text of course)

						<li>
							<a class="page-scroll" href="#features">Features</a>
						</li>

To make a menu item active, you have to add .active class to that item's li element.

						<li class="active">
							<a class="page-scroll" href="#page-top">Home</a>
						</li>

Header - Carousel

Header contains carousel and slider.

						<!-- Header Carousel -->
						<header id="myCarousel" class="carousel text-center">
							...
							...
						</header>
						<!-- End of Carousel -->

To change Slider images you have to put the images you want in images folder first. After that you have to change the image path for all slider pages:

						<!-- Here you can change the slider background -->
						<div class="fill" style="background-image:url('images/slider1.jpg');" 
						data-center="background-position: 50% 0px;" data-top-bottom="background-position: 50% -100px;" 
						data-anchor-target="#slide-1">

Wrapper

Wrapper contains Features, About US, Portfolio, Prices and Contact. It uses two mains classes: .container and .colored_container. You have two options: .container uses the default background and .colored_container uses custom colors.

						<!-- Page Content -->
						<wrapper>
							...
							...
						</wrapper>
						<!-- End of Page Content -->

To add a colored container you have to use the following:

						<div class="colored_container text-center" id="aboutus">
							<div class="container">
								...
								...
							</div>
						</div>

Circle Progress

We use javascript plugin Circle Progress to make progress charts. To learn how to use them just go here.

To create a circle just use the following code(example). You can change the settings on the evotemplate.js file. To learn how to do that, just use the link above.

							<div class="circles">
								<div id="circle_one">
									<div class="circle-txt"></div>
								</div>
							</div>

						</div>

Portfolio

To create the protfolio use the following code:

						<!-- Portfolio Section -->
						<div class="container" id="port">
							<!-- Portfolio Title -->
							<h1 class="page-header text-center wow animated fadeInDown">
								Recent Works
							</h1>

							<nav class="primary clearfix wow animated fadeIn">
								<ul>
									<li><a href="#" class="selected" data-filter="*">All</a></li>
									<li><a href="#" data-filter=".web">Web Design</a></li>
									<li><a href="#" data-filter=".ill">Ilustration</a></li>
									<li><a href="#" data-filter=".logo">Logo</a></li>
									<li><a href="#" data-filter=".video">Video</a></li>
									<li><a href="#" data-filter=".print">Print Design</a></li>
								</ul>
							</nav>
							<section class="main">
								<div class="portfolio">
									<article class="entry video wow animated fadeIn">
										<a data-rel="prettyPhoto" href="http://vimeo.com/34266952">
										<img class="image_port" src="images/portfolio/work1.jpg" alt="">
										 <span class="magnifier text-center"><div class="magnifier_icon"><i class="fa fa-2x fa-search"></i></div></span>
										</a>
									</article>
								</div>
							</section>
						</div> 
						<!-- End of Portfolio section -->

You can add as many items as you like with the article code:

						<article class="entry video wow animated fadeIn">
							<a data-rel="prettyPhoto" href="http://vimeo.com/34266952">
							<img class="image_port" src="images/portfolio/work1.jpg" alt="">
							 <span class="magnifier text-center"><div class="magnifier_icon"><i class="fa fa-2x fa-search"></i></div></span>
							</a>
						</article>

To change the category, just add watherver name you like to the article class, just after the word entry. You can use photos or videos. Just put the file url or web url on the href section.

Contact

To create the form contact just add:

						<!-- Contact -->
						<div class="container" id="contact">
							<!-- Contact Text -->
							<div class="col-md-12">
								<div class="col-md-7 col-lg-6" id="contact_form">
									<!-- Contact Title -->
									<div class="row  wow animated fadeInDown">
										<h2 class="page-header text-center">Contact US</h2>
									</div>
									<form name="sentMessage" id="contactForm" novalidate>
										<!-- Name -->
										<div class="row">
											<div class="control-group form-group">
												<div class="controls">
													<!-- Name Label -->
													<label class="wow animated fadeIn">Full Name:</label>

										</div>
										<!-- Phone Number -->

											<div class="control-group form-group">
												<div class="controls">
													<!-- Phone Number Label -->
													<label class="wow animated fadeIn" data-wow-delay="0.2s">Phone Number:</label>

												</div>

										</div>
										<!-- Email -->

											<div class="control-group form-group">

Change Theme Color

To change color, you have to change the following line:

						<!-- Custom Color CSS -->
						<link href="css/default.css" rel="stylesheet" class="color">

You can add your custom css color file or use the default ones, witch are located in css/colors.

For example:

						<!-- Custom Color CSS -->
						<link href="css/colors/style_green.css" rel="stylesheet" class="color">

If you do not want to use custom color, you can delete the custom color css line. The default color is included in styles.css file.

Fonts

EvoTeamplate uses Armata Regular for the h1, h3, h4, h5, h6 and the navbar. The rest of the page uses: "Helvetica Neue",Helvetica,Arial,sans-serif
